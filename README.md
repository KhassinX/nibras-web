# nibras-web

Public web property for **Nibras — Quran** · **nibras.khassinx.com**.

A reverent reader of the Qur'an in Arabic, with translations of the meaning, and an
on-device, non-generative concordance (it surfaces connections; it never interprets).
Privacy-first. For iPhone, iPad, Mac, and Apple Watch. Arabic text by Tanzil.net (CC BY 3.0);
English translation of the meaning by the Rowwad Translation Center and Spanish by Isa García, both
via QuranEnc.com / IslamHouse. Part of the KhassinX ecosystem. © KHASSINX LLC.
